Overview

A professional real-time automatic backup application built with Python.

This application continuously monitors selected folders and automatically backs up new or modified files using FIFO scheduling.

⸻

Features

* Real-time file monitoring
* FIFO backup scheduling
* Multi-threading
* Incremental backups
* Duplicate detection
* Modern GUI
* Live logs
* Background backup processing
* Pause / Resume / Stop controls
* Persistent settings
* Logging system

⸻

Technologies Used

* Python 3
* Tkinter / CustomTkinter
* Watchdog
* Queue
* Threading
* Logging
